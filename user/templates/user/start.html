Hackathon Rules

  • There will be no access to the internet during the first round.
  • Each team will be given 10 coding problems based on problem solving.
  • Based on the number of problems solved and the time taken, the teams will be awarded points.
  • Each Problem consists of 400 points and every 3 mins, the teams will lose 10 points.
  • For the last 30 mins every correct solution will be awarded 100 points.
  • The results and points for each question can be seen in the questions page.
  • The team can attempt the problem any number of times
  • The leaderboard with the total points will be displayed on the projector screen.
  • The screen will be logged out forcefully after 2hrs. Hence, make sure the code is submitted before time.
